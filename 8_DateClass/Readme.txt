Date.java program is written in Java 14.

This program defines a Date class that enables creation of Date objects. Date object can calculate the number of days between itself and another Date object, print the Date (YYYY/MM/DD), determine if it is a leap year, etc.

This program implements object-oriented programming concepts.

Algorithm to calculate the number of days between objects is based on the following: http://en.wikipedia.org/wiki/Julian_day.
